# py-gopro-sync
Python CLI to sync media files from GoPro cameras via wifi
py-gopro-sync

# Installation

 Pip
 ```shell
 pip install git+https://github.com/binary-signal/py-gopro-sync.git@0.1.0
 ```
 
Poetry
```shell
 poetry add "git+https://github.com/binary-signal/py-gopro-sync.git@0.1.0"
 ```
